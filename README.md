# E3 VBS Scripts

Сборник скриптов на VBScript для автоматизации работы с E3.series и связанных задач.

## Структура проекта

```
API_E3/           # Документация и интерфейсы API
archive/          # Архив старых скриптов
colors/           # Скрипты для работы с цветами
common/           # Общие скрипты для разных задач
  AddAttibuteValue.vbs
  clean_SIG.vbs
  ...
docs/             # Документация проекта
projects/         # Скрипты, сгруппированные по проектам
  BHK/
    ClearAllOOO.vbs
    ...
  GIGA/
    ClearAllOOO.vbs
    ...
relays/           # Скрипты для работы с реле
signals/          # Скрипты для работы с сигналами
templates/        # Шаблоны скриптов
utils/            # Вспомогательные скрипты (рисование, обработка)
wires/            # Скрипты для работы с проводами
```

## Примеры скриптов
- `common/mark.vbs` — маркировка элементов
- `projects/GIGA/Э3_SynchronizeOooPoo.vbs` — синхронизация объектов OOO и POO
- `utils/DrawSmile.vbs` — пример графического скрипта

## Как использовать
1. Откройте нужный скрипт в редакторе.
2. Запустите через задачу VS Code "Запуск текущего VBS скрипта" или вручную через cscript.
